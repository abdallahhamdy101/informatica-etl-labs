# Informatica PowerCenter ETL Practice

This repository contains hands-on ETL practice projects using Informatica PowerCenter.
It demonstrates core data integration concepts including mappings, transformations, workflows, and data loading.

## Tools
- Informatica PowerCenter
- SQL Server
- Flat Files
- Oracle

## Concepts Covered
- ETL Architecture
- Data Extraction
- Data Transformation
- Data Loading
- Data Validation
- Error Handling (basic)
- Data Quality

## Transformations Used
- Source Qualifier
- Filter
- Router
- Rank
- Aggregator
- Joiner
- Lookup
- Expression

## Project Structure
Each project contains:
- Mapping screenshots
- Workflow screenshots
- Successful session runs
